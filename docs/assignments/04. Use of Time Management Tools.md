# Uso de Ferramentas para Gerenciamento de Tempo

1) Reúna seu grupo do Projeto

2) Realize os seguintes artefatos e publique, individualmente, a imagem ou pdf dos seguintes artefatos:
- Estrutura Analítica do Projeto
- Gráfico de Dependências / Diagrama do Caminho Crítico
- Gráfico de Gantt

3) Apresentar e discutir os resultados, uso de slides
